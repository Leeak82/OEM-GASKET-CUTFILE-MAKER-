# 🛠️ OEM Gasket Cutfile Maker

Mobile gasket design tool with touch placement, SVG export, and camera-assisted workflow.

Built for mechanics, fabricators, and DIY builders who need fast, accurate gasket templates without expensive CAD software.

---

## 🚀 Live App

👉 https://leeak82.github.io/OEM-GASKET-CUTFILE-MAKER-/

---

## 📱 Features

- 🔧 Vehicle make/model selection
- 📐 Custom gasket sizing
- 🎯 Touch-based bolt placement
- 📷 Camera reference support
- 🧠 Preset + custom configurations
- 💾 Export to SVG (for Cricut, CNC, laser)
- 📄 Export/import JSON presets
- ⚡ Runs directly in browser (no install required)

---

## 🧰 Use Cases

- Cutting oil pan gaskets
- Head gasket mockups
- Custom fabrication
- CNC / laser cutting templates
- Cricut vinyl or gasket material cutting
- Field measurements and quick prototyping

---

## 🖥️ How It Works

1. Select vehicle make and model  
2. Adjust gasket dimensions  
3. Place bolt holes manually or via touch  
4. Preview gasket layout  
5. Export SVG for cutting
